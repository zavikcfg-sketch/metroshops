FunPay → группа сопровождения

При покупке бот пишет покупателю в чат FunPay и просит Player ID. После ответа с ID — карточка уходит в группу сопровождения. Добавьте бота в группу Telegram и выдайте право писать сообщения. Бот должен быть запущен.

Узнать ID: добавьте @userinfobot в группу или перешлите сообщение из группы боту @RawDataBot.
